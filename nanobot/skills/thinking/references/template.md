# Thinking Log Template

Use this template for each complex task in `memory/thinking.md`.

```text
## [YYYY-MM-DD HH:MM] <topic>
Problem:
Goal:
Constraints:
Known facts:

Thought 1:
Thought 2:
Thought 3:

Revision of Thought N: ... (optional)
Impact: ... (optional)

Branch A: ... (optional)
Branch B: ... (optional)
Comparison: ... (optional)

Conclusion:
Next action:
Confidence: low|medium|high
```

## Minimal Rules

- Append a new entry for each substantial task.
- Keep entries concise and factual.
- Do not create nested directories for thinking logs.
- Use `memory/thinking.md` as the default log file.
